Bluebird

Bluebird is a desktop based organisational tool designed for working parents on the go.

The intended target audience for Bluebird was mainly working parents, and their children. Our application helps both a parent and their child to manage important tasks schedules, and contacts. We consulted parents with young computer-savvy children for insights on our application before working on further features for Bluebird. We showed parents our first prototype which included the basic features of creating tasks/ schedules and organising contacts. Some insights we got from interviews with parents were:

- The child should not be able to edit heir parents’ schedules or tasks.

- The parent should be able to access their child’s schedules or tasks.

- The parent should be able to put important contacts in the child’s contact book.

- The tasks should be able to understand what their child keys in the command bar.

From there we went on to develop more features for our application based on the insights we received from our users. We had to ensure that the users we have in mind would find our app beneficial.

Here are the features that we've included after getting feedback from our users: (more details can be found in our user guide)

- A parental lock feature where only the parent knows the password.

This feature allows parents to set a password and restrict their child's activity within the app while it is in 'Child Mode' The child will no longer be able to delete any contacts, or access any features limited to the parent which is only accessible by entering a password.

The game is targeted at a casual audience and does not need any prior game experience. It is designed to be a tower defense game. However, the difference between our game and many others out there, is that this is a top-down 2D tower defense game. The player can control the in-game character to move around the map and defend the main objective from being attacked by the enemies.

EZCommute

This storyboard features the reasoning behind the idea for tis application. It portrays a confused user’s journey In an MRT station especially the interchange station with multiple MRT Lines (Downtown Line, Circle Line, North -East Line etc.).

The idea for the application was to provide users with a detailed information of the different lines within the palm their hands. The main intended audience for this application is tourists who are not familiar with how Singapore’s transport system works. Currently, there are a lot of existing applications that aims to achieve this. However, those applications do not show any detailed information of the Stations itself and the landmarks nearby.
